# Ongoing Development

We are an ongoing project, continuously working on bringing new features and improve the existing ones.

## Supported Orienteering Disciplines {/* #supported-orienteering-disciplines */}

As of now, we have support for the following orienteering disciplines:
- ✅ **Classic**: Foot-O / MTB-O / Ski-O (classification by time)
- ✅ **Score**: Rogaining / Ultra-Score (classification by points)
- 🚧 **Relay**
- ❌ **Raid**: Adventure Races

## Available Features {/* #available-features */}

We currently support the following features:
- ✅ Start list by classes
- ✅ Live results by classes
- ✅ Live online controls
- ✅ Live virtual tickets
- ✅ Live splits table
- ❌ Team results in FootO and Rogaining
- ❌ Individual courses (for example, one man relays)
- ✅ Overall championship results by classes
- ✅ Start list by clubs
- ❌ Championship start list by clubs
- ✅ Results by clubs
- ❌ Direct integration with online controls providers
(such as [ROC](https://roc.olresultat.se/) and [CPI](https://www.jaruori.es/))
